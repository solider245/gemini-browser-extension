## 📦 扩展已准备就绪！

## 🚀 安装步骤

### Chrome 浏览器
1. **打开 Chrome 浏览器**
2. **进入扩展管理页面**
   - 地址栏输入: `chrome://extensions/`
   - 或者：右上角三点菜单 → 更多工具 → 扩展程序
3. **启用开发者模式**
   - 右上角开启"开发者模式"开关
4. **加载扩展**
   - 点击"加载已解压的扩展程序"按钮
   - 选择 `gemini-browser-extension/dist` 文件夹
5. **完成安装**
   - 扩展会出现在扩展列表中
   - 浏览器右上角会显示扩展图标 🕒

### Edge 浏览器
1. **打开 Edge 浏览器**
2. **进入扩展管理页面**
   - 地址栏输入: `edge://extensions/`
3. **启用开发者模式**
   - 左下角开启"开发者模式"开关
4. **加载扩展**
   - 点击"加载已解压的扩展程序"
   - 选择 `gemini-browser-extension/dist` 文件夹

## 🔍 故障排除

### 如果扩展无法加载：
1. 确保选择了正确的 `dist` 文件夹
2. 检查浏览器是否支持 Manifest V3
3. 查看扩展管理页面的错误信息

### 如果数据不显示：
1. 确保已浏览一些网页
2. 尝试点击刷新按钮
3. 检查浏览器控制台是否有错误

### 如果界面显示异常：
1. 重新加载扩展
2. 清除浏览器缓存
3. 检查是否有权限问题
